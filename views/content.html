Missing permission to read and control devices

To use buttoninput you will have to give it permission to access your devices. Button Input currently does not have permission.

The Candle app store can give Button Input this permission. Visit the settings page for this addon via the Candle app store, and allow this addon to access your things. Then click save.

If you don't have the Candle App Store, go to Settings -> Developer and then Create Local Authorization. Generate a key there by clicking on Allow at the bottom of the page. Copy the token that's generated, and then visit the settings page for this addon. There you can paste in the code, and click Apply to save it.

Debugging is enabled

This addon is currently set to save detailed information about its operation in the internal logs. This may impact your privacy, as this also means that insights into your behaviour might be stored in those logs.

You can disable debugging mode by going to Settings, then Addons, and then clicking on the Settings button for the Button Input addon. There you can uncheck the debugging option and click Apply.
